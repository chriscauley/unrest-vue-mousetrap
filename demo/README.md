# @unrest/vue/demo

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development

Any changes to self documenting pages (Basics.vue, etc) will require stopping and starting the dev server.

```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
